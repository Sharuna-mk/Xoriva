# Xoriva E-Commerce Website

Xoriva is a full-stack MERN-based e-commerce platform designed to provide a seamless online shopping experience with modern UI, secure authentication, product management, cart system, and Stripe payment integration.

---

## Features

- Role-based UI for Admin and User with protected routes and conditional rendering
- Peer-to-peer thrift marketplace where users can list, browse, and purchase second-hand items
- Firebase Authentication for secure sign-in with email/password and social providers
- Stripe payment integration for seamless and secure checkout flow
- Google Gemini AI chatbot embedded for real-time user support and product assistance
- Global state management with Redux Toolkit, reducing redundant API calls by 35%
- Fully responsive design built with Tailwind CSS across all screen sizes
- Axios-based API layer with centralized request handling and auth token injection
